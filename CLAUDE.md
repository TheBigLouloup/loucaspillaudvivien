- The site is a **static HTML/CSS/JS site** (no frameworks, no build step) for GitHub Pages
- All data files are in `site-instructions/data/`
- The old site content is in `old_site/` for reference
- Build output goes at the repo root (index.html, css/, js/, etc.) alongside the existing old_site/ and site-instructions/ directories
- The photo (tortue.jpg) is at `site-instructions/img/photo.jpg`
- Presentations/Articles PDFs are NOT in the repo — link to them but they won't resolve until manually added
- `data/quotes.json` and `data/acknowledgments.json` have placeholder content — use what's there
- Do NOT attempt to scrape Google Scholar — use the pre-filled `data/papers.json`
- Use Cytoscape.js from CDN for the research graph
- Use Google Fonts from CDN for typography
